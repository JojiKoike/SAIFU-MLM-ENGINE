# Requirement

* Scala 2.13.2
* PlayFramework 2.8

# Installation

```bash
docker-compose up -d
```

# Usage

## Database

#### Migration

```sbtshell
flyway/flywayMigration
```

#### Migration Clear

```sbtshell
flyway/flywayClear
```

#### Migration Info

```sbtshell
flyway/flywayInfo
```

#### Migration Target DB URL

```sbtshell
flyway/flywayUrl
```
