# Requirement

* Scala 2.13.2

# Installation

```bash
docker-compose up -d
```

# Usage

## Database

#### Migration

```sbtshell
flyway/flywayMigrate
```

#### Migration Clear

```sbtshell
flyway/flywayClean
```

#### Migration Info
```sbtshell
flywa/flywayInfo
```

#### Migration Target DB URL
```sbtshell
flyway/flywayUrl
```

## Slick
#### Code Generation
```sbtshell
slickCodegen
```
